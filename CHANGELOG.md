# Changelog

## 0.1.0 — Initial Release

- Multi-phase audit pipeline: setup, static scan, reconnaissance, vulnerability scan, PoC construction, report
- Real-time web dashboard with SSE streaming
- CLI mode (`--no-ui`) for headless operation
- Reward-hack pipeline (`--hack-it`) for demonstrating exploits
- AI backends: Claude Code and OpenAI Codex
- Docker sandbox with `--network=none` isolation for static analysis
- Vulnerability classes V1–V8
- Static analysis via Semgrep, Bandit, Hadolint, Docker Analyzer, Trust Mapper
